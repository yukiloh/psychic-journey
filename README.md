#### 项目概述
一款社交问答的网站，用户进行话题或问题的发起，与其他用户进行讨论；
注册方面接入GitHub的OAuth认证，亦可在本地进行账号注册；
用户登陆融合了Shiro进行用户的账号权限认证；
提供个人用户问题发起功能，附件上传，针对问题进行回复；

项目演示：https://www.bilibili.com/video/av77093217
